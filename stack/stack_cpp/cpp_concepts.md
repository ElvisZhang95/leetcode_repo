### C/C++语言中i++与++1的区别

i++ ：先引用后增加，先在i所在的表达式中使用i的当前值，后让i加1

后置自增：

```c++
int a = 0;
int b = ++a;
// a=1, b=1
```



++i ：先增加后引用，让i先加1，然后在i所在的表达式中使用i的新值

前置自增：

```c++
int a = 0;
int b = a++;
// a=1, b=0
```

在for循环中两者没有区别；但是++i更快。因为编译器不需要保存一个临时变量保存i，当你做一个非常非常非常大的循环的时候，可能会体会到这一点速度差异
